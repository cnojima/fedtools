Defined in lib/yui3-utils.js:51

Clone the YUI3 git repository in a temporary folder. If a branch is specified, a shallow clone will happen. This command is verbose by design.

Parameters:

  • options Object
    • url String

      The YUI git repository URL.

    • [branch] String optional

      The branch to checkout.

  • done Function

    Callback to execute when done. It gets 2 arguments:

    • error String

      On success, this will be null.

    • srcPath String

      The temporary path to the cloned YUI3 repository.

  • verbose
  • pkgConfig
  • done
)
async

Defined in lib/yui3-utils.js:222

Helper method supposed to be called only for a command line direct call. It uses several public APIs that can be used independently.

It will:

  • Prompt the user for some options (location, branch)
  • Clone YUI3 in a temporary location
  • Copy YUI3 src to WF2 src

Parameters:

  • verbose Boolean

    True to log stderr/stdout.

  • pkgConfig Object

    Object holding package.json 'config' properties.

  • done Function

    Callback to execute when done. It gets 1 argument:

    • error String

      On success, this will be null.
