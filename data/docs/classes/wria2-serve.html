Defined in lib/wria2-serve.js:21

Method to be called directly by the main command line library to:

  • Extract a packaged WF-RIA2 tarball.
  • Run an internal JSP container to serve examples/tests/API.
  • Optionaly open the default browser with the corresponding URL.

Parameters:

  • debug Boolean

    True to display stderr/stdout in the console.

  • options Object
    • file String

      The path+filename of the WF-RIA2 tarball or the name of the already extracted folder.

  • done Function

    Callback to execute when done. It gets 2 arguments:

    • error String

      Not null if there is a problem with the command.

    • stdOut String

      The resulting output string without the trailing \n
